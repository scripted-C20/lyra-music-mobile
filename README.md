# Lyra Music Mobile

Lyra Music Mobile 是一个基于 React Native 的移动端音乐播放器项目。本仓库基于 LX Music Mobile 进行二次修改，主要面向 Android 端使用与学习交流。

项目地址：<https://github.com/scripted-C20/-lyra-music-mobile>

## 项目说明

- 基于 React Native `0.73.x` 开发，主要支持 Android 5.0 及以上设备。
- 保留 LX Music Mobile 的本地列表、在线搜索、歌单、排行榜、歌词、播放控制、数据备份、同步、自定义源等核心能力。
- 重构了移动端首页、歌单、排行榜、我的列表、设置、自定义源、播放胶囊、歌曲详情页等 UI 与交互。
- 移除了软件更新入口，但保留本地版本号展示。
- 当前仓库仅用于技术学习、界面实验和个人使用研究，请遵守所在地法律法规与音乐平台规则。

## 源地址说明

本项目的音乐检索、歌单、排行榜等在线数据逻辑来自原项目已有实现，数据源可能来自公开网络接口或用户自行导入的自定义源脚本。

- 原始项目：<https://github.com/lyswhut/lx-music-mobile>
- 桌面端参考项目：<https://github.com/lyswhut/lx-music-desktop>
- 本二改仓库：<https://github.com/scripted-C20/-lyra-music-mobile>

本项目本身不提供、不存储、不分发任何音乐音频文件。实际播放链接由用户选择的源返回，链接的可用性、准确性与合法性由对应源和使用者自行负责。

## 二次修改说明

本仓库在原项目基础上做了较多移动端定制，包含但不限于：

- 将应用视觉改为更偏网易云红主题的移动端风格。
- 重排顶部导航、页面操作区、列表卡片、底部播放胶囊与歌曲详情页。
- 为排行榜、歌单详情、搜索结果、我的列表等歌曲列表增加当前播放高亮与定位到当前播放按钮。
- 优化底部播放胶囊播放状态、封面旋转、播放/暂停控制与加载中暂停体验。
- 自定义源设置页单独成区，加入本地源、订阅源、导入、删除、测试、测试池与可停止测试等流程。
- 强化歌单链接解析与添加歌单交互，可根据友商等链接自动选择对应平台。。
- 将桌面歌词与状态栏/胶囊歌词拆成两套设置与悬浮窗实例，支持独立开关、颜色、背景、位置、尺寸和预设微调。
- 补齐 Android 标准 MediaSession / 媒体通知 / 耳机按键兼容，保留安全的播放、暂停、上一首、下一首、进度跳转等能力。
- 自定义源测试改为移动端内置闭环：本地源与订阅源内直接测试，使用 5 worker 测试池、2 路源请求并发、可停止测试和导入去重。
- 补齐基础设置、列表设置、主题色、歌词设置等可见设置项的落库与生效链路。
- 优化 Flyme/魅族状态栏歌词接入。
- Android 标准媒体通知、MediaSession、耳机按键等基础播放控制兼容。
- 禁用内置软件更新入口，仅保留本地版本展示。
- v1.8.7 继续修复搜索预建议链路：补齐多源 `tipSearch` 导出，预搜索失败时回退到真实搜索结果，并保持从首页进入搜索时清空、二级返回时保留的页面承接。
- v1.8.7 优化播放暂停闭环：加载/取链阶段也可暂停，暂停后停止当前取源任务并避免自动切到下一首，底部胶囊与歌曲详情页播放按钮状态保持同步。
- v1.8.7 将顶部搜索、搜索/歌单/排行榜/我的列表/设置操作区和歌单详情操作栏接入字体设置联动，文字、按钮高度、宽度、间距、圆角和图标尺寸随应用字体大小动态调整。

## 开发与打包

安装依赖：

```bash
npm install
```

启动 Metro：

```bash
npm run start
```

安装到 Android 设备或模拟器：

```bash
npm run dev
```

打包 Android Release：

```bash
npm run pack:android
```

打包产物通常位于：

```text
android/app/build/outputs/apk/release/
```
